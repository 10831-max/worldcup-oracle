# Security Policy

## Reporting a Vulnerability
Please report security vulnerabilities via GitHub Issues with the `security` label.
Do NOT disclose security issues publicly until they have been addressed.

## Supported Versions
| Version | Supported |
|---------|-----------|
| v2.x    | ✅ Active |
| v1.x    | ❌ End of life |

## Security Best Practices
- This project runs entirely in the browser
- No user data is sent to any server
- All computation is local
- localStorage is used for score persistence only
