# 🤝 Contributing to 2026 World Cup Analyzer

感谢你愿意为这个项目做出贡献！🎉

## 如何贡献

### 🐛 报告 Bug
如果你发现了 Bug：
1. 搜索 [Issues](../../issues) 确认没有被报告过
2. 使用 Bug Report 模板创建新 Issue
3. 描述复现步骤、预期行为和实际行为

### 💡 功能建议
有好的想法？使用 Feature Request 模板告诉我们！

### 🔧 提交代码
```bash
# 1. Fork 本仓库
# 2. 克隆到本地
git clone https://github.com/你的用户名/yz_football.git
cd yz_football

# 3. 创建分支
git checkout -b feature/my-awesome-feature

# 4. 修改代码（确保兼容 file:// 协议和现代浏览器）

# 5. 提交
git add .
git commit -m "feat: 添加了xxx功能"

# 6. 推送并发起 PR
git push origin feature/my-awesome-feature
```

### 📸 贡献球队照片
1. 找到高清球队/国旗图片
2. 压缩到 <500KB
3. 重命名为 `{3字母球队ID}.jpg`（如 `BRA.jpg`）
4. 放到 `img/teams/` 文件夹
5. 提交 PR

### ✅ 代码规范
- 纯 JavaScript (ES6+)，无框架
- 兼容 `file://` 协议（本地双击打开）
- 支持 Chrome/Firefox/Edge/Safari 最新版
- 注释使用中文

## 📋 Commit 规范

使用 [Conventional Commits](https://www.conventionalcommits.org/)：
- `feat:` 新功能
- `fix:` 修复Bug
- `docs:` 文档更新
- `style:` 样式修改
- `refactor:` 代码重构
- `perf:` 性能优化
