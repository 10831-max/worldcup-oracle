# Contributing to worldcup-oracle

Thank you for your interest in contributing! 🎉

## How to Contribute

### Bug Reports
1. Search existing [Issues](../../issues) first
2. Use the Bug Report template
3. Include reproduction steps, expected vs actual behavior

### Feature Requests
Use the Feature Request template to propose new ideas.

### Code Contributions
```bash
git clone https://github.com/10831-max/worldcup-oracle.git
cd worldcup-oracle
git checkout -b feature/my-awesome-feature
# Make changes (ensure file:// protocol compatibility)
git add . && git commit -m "feat: description"
git push origin feature/my-awesome-feature
# Open a Pull Request
```

### Adding Team Photos
1. Find a high-quality team/flag image
2. Compress to <500KB
3. Name as `{3-letter-ID}.jpg` (e.g., `BRA.jpg`)
4. Place in `img/teams/`
5. Submit a PR

### Code Standards
- Pure JavaScript (ES6+), zero frameworks
- Compatible with `file://` protocol
- Supports Chrome/Firefox/Edge/Safari latest
- Use Conventional Commits (`feat:`, `fix:`, `docs:`, etc.)

Thank you! ❤️
